# MVC Next.js Application

A Next.js application using **Model-View-Controller (MVC)** architecture with a **CSV-based mock database** for managing political promises and campaigns.

## Prerequisites

- Node.js 18 or higher
- npm or yarn package manager

## Installation

1. **Install Dependencies**

   ```bash
   npm install
   ```

2. **Run the Development Server**

   ```bash
   npm run dev
   ```

3. **Access the Application**

   Open [http://localhost:3000](http://localhost:3000) in your browser
